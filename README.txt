INSTRUCCIONES
Ejecute el programa y presione las teclas que se le indica. 
La interacción con el mouse consiste en, primer click sera la posición inicial y mantienes el click arrastrando hasta el siguiente punto
donde sueltas el click, posteriormente se creará la linea o el círculo.

Las teclas disponible son A,B,C,F y M. Siendo la M el regreso al menú.
Como indica en el menú: 
	A -> Linea DDA
	B -> Linea Bresenham
	C -> Círculo sin relleno
	F -> Círculo con relleno
	M -> Vuelta al Menú

Las imagenes necesarias (veáse la carpeta con el mismo nombre) deben ser puestas en la carpeta res (como indica la práctica).
